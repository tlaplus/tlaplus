TLA+ Resources

The best place to begin learning about TLA+, its tools, and the Toolbox is in the TLA+ Hyperbook, which is only partially written.  The textbook and reference manual for the TLA+ language is Specifying Systems.  Changes to the language since that book was written are described here.  The latest news about the TLA+ tools and the Toolbox can be found on the home page of the TLA+ community.
