Getting Started

The first thing you see when you start the Toolbox for the first time is the Welcome View.  If you are new to the Toolbox, you should read the Using Help and Managing Views help pages.

From the welcome view, you will usually open a specification.  See the Managing Your Specifications page.  You can also set the Toolbox Preferences.

For further information about TLA+ and the TLA+ tools, see the TLA+ Resources help page.  When using the Toolbox, TLA+ means TLA+2, the revised version of the language; see the TLA+2 web page.

Important: The In Case of Trouble help page tells you what to do if you encounter a Toolbox bug. In the unlikely event that a bug prevents you from reading the help pages, these pages are available on the Web at http://tla.msr-inria.inria.fr/tlatoolbox/doc/contents.html.  Bookmark this URL in your browser for emergence use.
